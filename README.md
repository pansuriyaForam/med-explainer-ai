# Medicine Explainer AI

AI-powered health literacy assistant for explaining medicines in simple language.

## Stage 1 MVP

- Manual medicine name entry
- No OCR
- No prescription scanning
- No diagnosis
- Simple explanation of medicines
- Indian language translation planned

## Tech Stack

- FastAPI backend
- Streamlit frontend
- Python

## Run Backend

```bash
uvicorn backend.main:app --reload
```

## Run Frontend
```
streamlit run frontend/app.py
```
